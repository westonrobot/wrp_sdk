Detailed Description

Satellite Navigation Fix message If the covariance of the fix is known, fill it in completely. If the GPS receiver provides the variance of each measurement, put them along the diagonal. If only Dilution of Precision is available, estimate an approximate covariance from that.

Position covariance[m ^ 2] defined relative to a tangential plane through the reported position.
